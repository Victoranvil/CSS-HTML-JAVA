Aprendendo HTML

Esse são os primeiros passos para aprender a criar uma página em HTML.

HTML é uma linguagem de marcação de hipertexto que define a estrutura e o significado do conteúdo de uma página web. É o componente base da web e é usado para criar páginas web e aplicações online. Não é considerado uma linguagem de programação. Trabalha em conjunto com o CSS, pode ser editado por qualquer editor de texto.
